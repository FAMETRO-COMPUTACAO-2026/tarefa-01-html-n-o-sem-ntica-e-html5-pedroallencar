Exercício 01: Do Caos à Estrutura – Semântica HTML5
🎯 Objetivo
Nesta tarefa, o teu objetivo é transformar um site com estrutura genérica (baseado apenas em <div>) num documento HTML5 Semântico. A aparência visual não deve mudar, mas o "significado" do código para motores de busca e leitores de ecrã deve ser profissionalizado.

🛠️ O Problema
O ficheiro index.html atual funciona, mas é semanticamente "mudo". Ele utiliza id e class para definir as áreas, o que não é uma boa prática na Web moderna.

🚀 Tarefas a Realizar
Identificar e Substituir:

Substitui a <div id="header"> pela tag adequada: <header>.

Envolve os links de navegação na tag <nav>.

Define a área principal de conteúdo com <main>.

Cada post do blog deve ser transformado num <article>.

A barra lateral deve ser um <aside>.

O rodapé deve utilizar a tag <footer>.

Hierarquia de Títulos:

Garanta que o título principal do site use <h1>.

Os títulos dos artigos devem usar <h2> ou <h3> de forma lógica.

Acessibilidade:

Define o idioma da página na tag HTML: <html lang="pt-pt"> ou pt-br.

Se adicionares imagens, não te esqueças do atributo alt.
